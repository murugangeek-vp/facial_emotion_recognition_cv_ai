💡 **How it works (Simple):**

• Capture face using camera
• Detect facial features (eyes, lips, expressions)
• Apply trained ML model
• Predict emotion in real-time

👉 This is called **Emotion AI** — where machines can read, interpret, and respond to human feelings.
